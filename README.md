# crafting-bedrock
using beacons you can craft bedrock
 this is a pack for 1.18
# only in 1.18
Guide to put this pack on minecraft
1. open minecraft on 1.18
2. start the game and press esc
3. in this menu press options and after <pres resource packs>
4. now you upload the pack and press done
5. now press done and back to game
